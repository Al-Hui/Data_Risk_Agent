# Data Risk Agent v1

Приложение собирает кандидаты на риск данных из технологических инцидентов, ролей ИТ-услуг в бизнес-процессах, контрактов на поставку данных, уже зарегистрированных рисков и данных по remediation works.

В демо-режиме вместо реального LLM используется детерминированный провайдер ответов из `data/demo_reasoning.json`, чтобы можно было стабильно показать работу пайплайна и дашборда.

## Что умеет

- пакетно анализирует таблицу инцидентов
- различает стандартные и нестандартные кейсы
- извлекает идентификаторы контрактов и оферт
- определяет участие ИТ-услуги в бизнес-процессах
- формирует сценарии риска и агрегирует их по `бизнес-процесс + ИТ-услуга`
- валидирует риск и предлагает митигации
- сравнивает с уже существующими рисками
- показывает review dashboard в Streamlit
- умеет подготовить запись риска и мероприятий во внешний реестр через API-клиенты

## Запуск

```bash
python3 -m pytest
streamlit run streamlit_app.py
```

## Демо-данные

В каталоге `data/` лежат CSV-файлы, на которых приложение можно запустить без дополнительной настройки.
